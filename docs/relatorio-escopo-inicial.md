# Proposta inicial do trabalho individual final

## Aplicação empresarial para processos internos 

A escolha do tema justifica-se por um conjunto de fatores, dentre os quais se destacam o interesse acadêmico em desenvolver um sistema mais completo e robusto, bem como o interesse prévio em construir uma aplicação desse escopo para compor portfólio. Ademais, a proposta apresenta diversas possibilidades de aplicação prática dos requisitos do trabalho.

A ideia central consiste no desenvolvimento de um sistema centralizado voltado aos processos internos de uma empresa genérica “X”, no qual será possível configurar o ambiente e disponibilizar aos funcionários um painel (dashboard) de trabalho.

Inicialmente, o funcionário deverá realizar login na plataforma (com credenciais fornecidas pelo gerente ou administrador do sistema). Na dashboard principal, será possível visualizar as tarefas atribuídas individualmente ou em equipe (quando o usuário fizer parte dela). Haverá, ainda, uma aba destinada à comunicação, com um sistema de conversas entre usuários ou grupos, e, por fim, uma funcionalidade para registro de ponto diretamente no sistema.

Determinados funcionários terão acesso a abas específicas, como: sistema de gestão de estoque; sistema de controle de caixa (possivelmente em versão simplificada, a fim de reduzir a complexidade do trabalho); e sistema de gestão administrativa, contemplando cadastro de funcionários, definição de permissões, entre outros aspectos. Por padrão, apenas o administrador terá acesso a essa área, embora cargos de gerência também possam ser autorizados. Outras funcionalidades poderão ser adicionadas conforme a necessidade.

O sistema contará com armazenamento próprio de dados. A implementação dependerá das limitações disponíveis: o trabalho permite o uso de arquivos `.txt`, porém, se viável, será utilizado um banco de dados integrado.
